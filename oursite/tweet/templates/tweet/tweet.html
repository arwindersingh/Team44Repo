Here, tweets data is collected by Twitter API and make sentiment anaylysis to classfy them into positive, negative and neutral.
Sentiment Analysis: In our project, we applied the VADER ((Valence Aware Dictionary and Sentiment Reasoner) tool in nltk library to achieve the sentiment analysis function. VaderSentiment is a fully open sourced lexicon and rule-based sentiment analysis tool that is specifically attuned to sentiments expressed in social media. The time complexity of the algorithm is O(N). The VADER sentiment lexicon is sensitive both the polarity and the intensity of sentiments expressed in social media contexts, and is also generally applicable to sentiment analysis in other domains.
